# Keybind Core (ML4W parity)

MOD = SUPER (Hyprland) / CMD (macOS)
Physical MOD key = **Caps Lock** (tap Esc, hold Super)

Focus:
- MOD + H/J/K/L  -> focus left/down/up/right

Move window:
- MOD + SHIFT + H/J/K/L -> move left/down/up/right

Workspaces:
- MOD + 1..9 -> switch workspace
- MOD + SHIFT + 1..9 -> move window to workspace

Apps:
- MOD + Enter -> terminal (kitty)
- MOD + D     -> launcher (Alfred / rofi)
- MOD + Q     -> close window

Help:
- MOD + /     -> show cheat sheet overlay
